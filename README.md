# LightStudio Classic

    北京航空航天大学软件学院2022级面向对象程序设计大作业

软件说明见 [*LightStudio Classic Wiki*](https://github.com/yranium2023/LightStudioClassic/wiki)

这是一个基于JavaFX框架开发的类似Adobe Lightroom Classic的照片管理、编辑软件
